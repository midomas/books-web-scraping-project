# Multithreaded Web Scraping & Data Analysis

## Project Overview
This project demonstrates a real-world data science application using:
- Web Scraping
- Multithreading
- JSON File Manipulation
- Data Preprocessing & Analysis with Pandas

Data is scraped from **books.toscrape.com**, cleaned, analyzed, and saved for further use.

## Tools & Technologies
- Python
- BeautifulSoup
- Requests
- Pandas
- Threading
- Jupyter Notebook

## How to Run
1. Open the notebook `books_scraping_clean.ipynb`
2. Run all cells from top to bottom
3. Outputs:
   - `scraped_books.json`
   - `processed_books_data.csv`

## Features
- Multithreaded scraping for faster data collection
- Safe preprocessing to handle encoding issues
- Clean and structured analysis
